v0v.cc: a URL Shortener Service

This is a URL shortener service written in Go. It's for my personal use and not open to public.

For source code, see here

API

To shorten a URL, send a POST request with it as the url field in the body. For example, to shorten https://google.com, run:

$ curl -d "url=google.com&auth=XXX" v0v.cc
where the output is v0v.cc/BST

auth is the authentication code needed.
